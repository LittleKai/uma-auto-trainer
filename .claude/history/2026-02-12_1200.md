# Change Log - 2026-02-12 12:00

## Summary
Added inline Move Up/Down arrow buttons to each preset item in the Preset Dialog, and changed the edit pencil icon color.

## Changes Made

### `gui/dialogs/preset_dialog.py`
- Removed bottom "Move Up" / "Move Down" buttons
- Added inline arrow buttons (▲ ▼) on the right side of each preset item
  - Gray (#888888) by default, turns blue (#0066CC) on hover
  - `_move_preset_up_by_num(preset_num)`: Moves specific preset up
  - `_move_preset_down_by_num(preset_num)`: Moves specific preset down
- Restructured `_create_preset_frame` layout: content_frame (left) + move_frame (right)
- Added `_set_frame_bg()` recursive helper for background color updates (handles deeper nesting)
- Simplified `_select_preset` to use `_set_frame_bg` instead of manual 2-level iteration

### `gui/tabs/event_choice_tab.py`
- Changed edit pencil icon (✎) color from blue (#0066CC) to gray (#888888)

## Testing Notes
- Click ▲/▼ arrows on any preset to swap it with the adjacent preset
- Preset data (name, Uma, cards, caps, style, stop conditions) all swap correctly
- Current preset "(Current)" indicator follows the data, not the slot
- Arrow hover effect: gray → blue
